# QR Code

Simple script to generate a QR code image from a URL using the `qrcode` Python package.

Usage

1. Create and activate a Python virtual environment (optional but recommended):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
python -m pip install --upgrade pip
pip install -r requirements.txt
```

3. Run the script:

```powershell
python main.py
```

The generated QR code will be saved to `D:\Coding\QR Code\qrcode.png` by default.

Files

- `main.py` — main script
- `requirements.txt` — Python dependencies
- `.gitignore` — common ignores for Python projects
